**Project is going to have the following components:**

1. 3.7V single-cell LiPO battery

2. TP4056 LiPO charging module

3. ESP32-C3 Supermini

4. MAX98357A Speaker Driver + 8Ohm Cheapo Speaker

5. WS2812B LED Ring.

6. MicroSD Card Reader Adapter + MicroSD Card

7. Other assorted hardware including 1 button, power toggle switch and of course the 3D printed housing + minifigure.
